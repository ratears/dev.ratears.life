分布式架构下常见问题

  • 集群部署，JVM锁解决不了并发问题
  • 远程调用，Spring事务失效了
  • 数据量太大，单库单表支撑不了业务压力了
  • 分库分表，数据库自增id重复了
  • 已经登录过了，访问另外一个服务还需要重新登录
  • 集群部署，Spring任务调度执行重复了

扣减库存实战

  • 需求背景：电商项目中，用户购买商品后，会对商品的库存进行扣减
  • 需求实现：根据用户购买商品及购买商品数量，对商品的库存进行指定数量的扣减。
  • 库存扣减项目技术点
    • Spring Boot 3.0
    • JDK17
    • Mybatis
    • Lombok


使用JMeter压力测试工具

  • Jmeter介绍
    • ApacheJMeter 是 Apache 组织基于Java 开发的压力测试工具，用于对软件做压力测试。
  • Jmeter用来做什么
    • 接口测试
    • 性能测试
    • 压力测试

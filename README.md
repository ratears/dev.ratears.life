



这是 ratears' blog 的源文件。

使用方式：

```git
git clone -b source https://github.com/ratears/dev.ratears.life.git

cd dev.ratears.life

cnpm install
# 安装部署插件
cnpm install hexo-deployer-git --save
cnpm install hexo-server --save

cd themes/keep
cnpm install

cd ../
hexo clean && hexo g && hexo d && hexo s


```
